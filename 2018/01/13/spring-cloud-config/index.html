高可用

现在我们的 Config Server 是一个单点的服务，一旦挂掉后面的 Config Client 将无法使用。所以在生成中，Config Server 必须是高可用的，那怎么做呢？很简单，跟我们前面讲的 Eureka 的高可用一样，只需要将同一个 Config Server 多部署几个即可。
